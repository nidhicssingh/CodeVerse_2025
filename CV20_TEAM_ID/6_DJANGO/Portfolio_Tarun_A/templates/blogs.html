Building a Computer from First Principles: My Nand2Tetris Journey

July 18, 2025 by Tarun

One of the most rewarding challenges I've undertaken is the Nand2Tetris course. The experience of building a complete, modern computer from the ground up, starting with a single NAND gate, has been incredibly insightful.

In the first five projects, I moved from elementary logic gates to building an Arithmetic Logic Unit (ALU), and finally, a fully functional CPU. Writing these components in a Hardware Description Language (HDL) provided a deep, practical understanding of computer architecture that goes far beyond theory. It's one thing to know what a CPU does; it's another to have built one yourself.

My First Step Into Web Development: A Simple HTML Page

June 25, 2025 by Tarun

Every developer's journey starts somewhere. For me, a significant first step was creating a basic, single-page portfolio using just HTML and CSS. It wasn't complex, but it was a powerful moment seeing my own code render a functional webpage in the browser for the first time.

This simple project was the catalyst for my passion for web development. It taught me the fundamentals of web structure and styling and motivated me to dive deeper into more advanced topics like JavaScript, Bootstrap, and eventually, the Django framework that powers this very site. It's a great reminder that even the most complex systems are built upon simple, solid foundations.
