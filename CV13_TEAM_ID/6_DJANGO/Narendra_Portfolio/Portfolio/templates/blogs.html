How I Fell in Love with Web Development

From Curiosity to Passion

It all started with curiosity — a simple desire to understand how websites actually work. I’d always seen the internet as a user, but something clicked the first time I opened up a code editor and saw my changes reflected live in a browser. HTML felt like magic, CSS like an art form, and JavaScript added that spark of interactivity. Before I knew it, I was hooked.

Building My First Projects

My first real project was far from perfect — but I remember how proud I felt when I built a responsive layout that didn’t break on mobile. I started exploring more advanced tools like Flask for Python-based web apps, AJAX for single-page behavior, and even authentication with Google login. These weren’t just technical wins — they were milestones that showed me how powerful and fun building for the web could be.

Why I Love It

Web development lets me blend logic with creativity. It's both structured and free-flowing — a space where problem-solving meets design. I can create tools that work, websites that move, and experiences that people interact with. That feeling when everything finally comes together — it’s addictive.

What I’m Working On Now

Currently, I’m exploring modern UI/UX principles, single-page applications, and cloud deployment using platforms like AWS and GitHub. I’ve also built a photography portfolio website with a blog (yes, the one you're reading!), styled to reflect who I am — both technically and creatively.
